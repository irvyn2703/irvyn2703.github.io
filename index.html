Identificador de leucemia
Facultad de ciencias de la computación BUAP
Desarrollo de un programa en Python para la identificación de leucemia mediante el entrenamiento de una red neuronal convolucional. Para este fin, se utilizó una biblioteca con una amplia variedad de imágenes que representan diferentes variantes de la leucemia.Además, antes del proceso de entrenamiento, las imágenes fueron sometidas a un procedimiento para obtener sus Regiones de Interés (ROI), que representan el área de la célula infectada con mayor precisión.
Control inteligente de dispositivos
Desarrollo de un sistema de control inteligente para habitaciones que ofrece funcionalidades avanzadas para regular la iluminación de forma automática o manual. El sistema utiliza sensores para detectar y ajustar la intensidad de las luces según la luminosidad ambiente y la actividad en la habitación.Además de la regulación automática, el sistema permite a los usuarios controlar las luces de manera manual a través de un punto de acceso que muestra una página web interactiva.
Procesamiento de imagenes
Aplicación gráfica integral que ofrece una amplia gama de herramientas para la edición avanzada de imágenes. Esta aplicación permite realizar desde funciones básicas, como la conversión a escala de grises, zoom y rotación, hasta operaciones más complejas como la aplicación de filtros utilizando la transformada de Fourier y operaciones de convolución.
